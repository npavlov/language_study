# Internationalization (i18n)

**File**: `src/js/i18n.js`

## Overview

All user-facing UI text is in Russian. English and Serbian are study languages only. Text is never hardcoded in mode files — always imported from `i18n.js`.

## Imports

```js
import { t, langLabel } from '../i18n.js';
```

## `langLabel(lang)` Function

```js
langLabel('en') → 'Английский'
langLabel('sr') → 'Сербский'
langLabel('ru') → 'Русский'
langLabel('fr') → 'FR'  // fallback: uppercase code
```

## Rules

1. **Never** hardcode Russian/English text in mode files
2. Always use `t.key_name` for UI strings
3. Always use `langLabel(lang)` for language name display
4. New strings must be added to `i18n.js` first, then referenced
